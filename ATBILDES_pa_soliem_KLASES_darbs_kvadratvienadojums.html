Vienādojumu piemēri pēc diskriminanta vērtības

  • 4 piemēri ar divām racionālām saknēm (D > 0, pilnīgs kvadrāts)
  • 4 piemēri bez reālām saknēm (D < 0)
  • 4 piemēri ar vienu sakni (D = 0)
a) 2x² - 8x + 8 = 0
1. solis – Nosaki koeficientus

a = 2, b = -8, c = 8

2. solis – Aprēķini diskriminantu

D = (-8)² - 4·2·8 = 64 - 64 = 0

3. solis – Sakņu formula

x = -(-8) / 2·2

4. solis – Atrisinājums

x = 2

b) x² + 6x + 13 = 0
1. solis – Nosaki koeficientus

a = 1, b = 6, c = 13

2. solis – Aprēķini diskriminantu

D = 6² - 4·1·13 = 36 - 52 = -16

3. solis – Sakņu formula

x = (-6 ± √-16) / 2·1

4. solis – Atrisinājums

Diskriminants ir negatīvs – reālu sakņu nav. x ∈ ∅
